Through out the units, you'll be working on building out a browser based **Memory Game**. While you'll be using the in-built editor for the some other exercises in this course, for the Memory Game specifically, you'll be using Sublime Text Editor 3 and then submitting your code via Github at the very end of the course. To get started, let's take a look at the tools we'll need to start building out your very own webpage.
